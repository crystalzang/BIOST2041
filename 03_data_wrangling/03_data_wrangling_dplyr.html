IV. Summarize Function

Example: Identify the average surgery hour for each race group

surgery_data%>%
  group_by(race)%>%
  summarize(count = n(),
            hour_mean = mean(hour),
           hour_median= median(hour),
           hour_sd = sd(hour))%>%
  mutate(perc = count/sum(count) * 100)
